Game1: Make and Escape

Designed by Hung-yu Chen

"Make and Escape" is a game including collecting the material, making them into tools, and escaping from the yard. The player needs to think a little bit to figure out how to escape!

Gameplay

The player controls the position of the main character with the arrow keys.

When the main character reaches the edge of the screen, the screen transitions to the adjacent screen (if it exists).

If the main character is standing next to a object that can interact with, the object is highlighted, and the player can press 'z' to interact with the object. The form of interaction could be pick it up, put sometiong on it, use it, and/or show a message window.

World

The world is basicly the yard, divided into three sceens and a winning screen.

Materials, Tools and Items

The main character can only carry one material, tool, or item at a time. Materials can be made into tools at the workbench, tools can be used to get the items, and items can help you escape from the yard.

Materials:

  • boards
  • rope
  • pickaxe head
  • stick
  • knife
  • rod

Tools:

  • bridge
  • pickaxe
  • knife with a long handle

Items:

  • crystal
  • coin
  • apple
  • rock
  • key

Workbench

Tools can be made at the workbench when the required materials are placed on it.

  • boards + rope = bridge
  • pickaxe head + stick = pickaxe
  • knife + rod = knife with a long handle

Usages of Tools

The bridge is used to cross the water to get the crystal.

The pickaxe is used to dig out the coin.

The knife with a long handle is used to get the apple from the tree.

Usage of the items

When crystal, coin, apple, and rock are placed on the four outer pillars correctly (right: coin, up: apple, left: crystal, down: rock), the key shows up on the center pillar.

The key can be used to open the door of the yard so that the main character can escape!

Hints

There are some hints shown in the message for the player to detremine the correct pillar to place the items.

  • The pond shows "I left without a trace." => crystal on the left pillar
  • The map shows "I'm always right." => coin on the right pillar
  • The tree shows "Look up!" => apple on the upper pillar
  • The scale shows "Settle me down." => rock on the lower pillar
